Production branch

    master

Development branch

    develop

Branch naming convention.

    master
    hotfix-name_of
    version-name_of
    feature-name_of
    release-name_of
    support-name_of
    lab-name_of
Branch names must be in all lowercase.

Danger zone

Never commit to the 'master' or 'development' branch. Always branch off of them and merge back to them.
